# Video to LLM Prompt Packager

A simple browser-based local automation utility to extract audio from video clips, transcribe it with blazing speed using Groq's cloud-hosted Whisper-large-v3, and wrap it smoothly inside structural templates.

## Prerequisites
1. Ensure your system has Python 3 and FFmpeg installed. On Linux Mint:
```bash
sudo apt update && sudo apt install python3 python3-venv ffmpeg -y
```
2. Get the Groq API key and save it somewhere (register on groq.com, go to API Keys at the top menu, create API key)
